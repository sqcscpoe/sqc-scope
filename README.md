# SQC Scope — QC Agent for Solar Installations

Automated Quality Control reports for Palmetto LightReach solar installations.

## Stack
- Next.js 14 App Router + TypeScript
- Claude claude-sonnet-4-5 vision API
- Google Drive API (Service Account)
- Resend email API
- TailwindCSS

## Setup
1. Copy `.env.example` → `.env.local` and fill in credentials
2. `npm install`
3. `npm run dev`
